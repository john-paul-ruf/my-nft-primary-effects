# my-nft-primary-effects

A plugin pack of 20 primary visual effects for NFT generative art, built for the [my-nft-gen](https://www.npmjs.com/package/my-nft-gen) framework.

## Effects

Each effect ships with 3 presets offering different visual intensities and animation styles. Every effect is animated and supports fuzz (accent, blur, and feather) post-processing.

## Installation

```bash
npm install
```

Requires `my-nft-gen` v1.6.9+ as a dependency.

## Usage

### As a Plugin

Register effects with the `my-nft-gen` framework:

```js
import { EffectRegistry, PositionRegistry } from 'my-nft-gen';
import { register } from 'my-nft-primary-effects';

await register(EffectRegistry, PositionRegistry);
```

### Direct Import

```js
import { FractalDendriteEffect, FractalDendriteConfig } from 'my-nft-primary-effects/src/index.js';

const config = new FractalDendriteConfig({});
const effect = new FractalDendriteEffect({ config });
```

## Test Rendering

Render effects to verify they work correctly:

```bash
# Show help
node scripts/testRender.js --help

# Test all effects
node scripts/testRender.js --all

# Test a single effect
node scripts/testRender.js --effect fractal-dendrite

# Test all effects with custom resolution and save frames
node scripts/testRender.js --all --frames 25 --width 1080 --height 1920 --save-frames --verbose
```

## Creating a New Effect

Use the template at `templates/PrimaryEffectTemplate.js` as a starting point:

1. Copy the template to `src/effects/primaryEffects/YourEffect/`
2. Rename and implement `YourEffectEffect.js` and `YourEffectConfig.js`
3. Export from `src/effects/primaryEffects/index.js`
4. Add presets to `src/effects/presets.js`
5. Register in `plugin.js`
